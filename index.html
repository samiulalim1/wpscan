Wordpress Scanner

This is an online WordPress Scanner for detecting and reporting WordPress site.

This site detect almost 90% acutestluy detecting and reporting WordPress site.

Scan : Upload or write the site link in here. Then click "Scan Now" to start scanning.

WordPress : Wordpress site link will show here. Which link is 100% WordPress those shown here.

NonWordPress : NonWordPress site link will show here. Which link is 90% NonWordPress those shown here.

Invalid : Invalid site link will show here. Which line has no link or invalid link those shown here.

Errors : Errors site link will show here. Which line can not scanned properly those shown here. This link could be WordPress or Not WordPress.

Problem : When you disconnect to internet or server got any error then Errors link show here.

Note : Wordpress site can found in errors.
